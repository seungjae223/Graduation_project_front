# 💰 여행 경비 기반 동선 최적화 여행 추천 시스템 (Frontend)

> 사용자의 여행 예산을 기반으로 현실적으로 여행 가능한 장소를 추천하고 효율적인 여행 동선을 제공하는 웹 서비스

---

## 📌 프로젝트 소개

본 프로젝트는 사용자가 입력한 **여행 예산, 여행 기간, 인원 수**를 기반으로  
예산 범위 내에서 여행 가능한 장소를 추천하고 **효율적인 여행 동선**을 제공하는 **경비 중심 여행 추천 웹 서비스**이다.

기존 여행 서비스는 목적지 중심의 정보를 제공하지만,  
본 서비스는 다음과 같은 질문에 답하는 것을 목표로 한다.

> **"이 예산으로 어디까지 여행할 수 있을까?"**

또한 추천된 여행지 내에서 **효율적인 여행 동선**을 제공하여  
사용자가 보다 합리적인 여행 계획을 세울 수 있도록 돕는다.

---

## 🎯 프로젝트 목적

- 여행 계획에서 가장 중요한 요소인 **예산 기반 의사결정 지원**
- 예산 초과 없는 **현실적인 여행지 추천**
- 추천 여행지 내 **효율적인 여행 동선 제공**
- React 기반 **프론트엔드 웹 서비스 구현 경험**

---

## 🧠 주요 기능

### 1️⃣ 여행 정보 입력

사용자는 여행 계획에 필요한 정보를 입력한다.

- 총 여행 예산
- 여행 기간 (일수)
- 인원 수
- 여행 스타일 (선택)

---

### 2️⃣ 여행 경비 계산

입력된 데이터를 기반으로 여행 경비를 계산한다.

경비 구성

- 교통비
- 숙박비
- 식비
- 관광비

여행 기간과 인원 수에 따라 **비용 가중치**가 적용된다.

---

### 3️⃣ 여행지 추천

입력된 예산 범위 내에서 여행 가능한 장소를 추천한다.

추천 기준

- 여행 예산
- 여행 기간
- 예상 여행 경비

추천 결과

- 국내 여행지
- 해외 여행지
- 예상 총 여행 경비 제공

---

### 4️⃣ 여행 동선 최적화

추천된 여행지 내 관광지를 기반으로  
**효율적인 여행 동선**을 제공한다.

- 이동 거리 최소화
- 시간 효율적인 관광 경로 추천

---

### 5️⃣ 추천 결과 UI 제공

추천 여행지는 **카드 UI 형태**로 제공된다.

카드 정보

- 여행지 이름
- 예상 총 여행 경비
- 예산 사용 비율
- 간단한 여행 정보

---

## 🖥️ 화면 구성

### 메인 페이지

여행 정보 입력 화면

입력 항목

- 여행 예산
- 여행 기간
- 인원 수
- 여행 스타일

---

### 추천 결과 페이지

예산에 맞는 여행지 목록 제공

- 여행지 카드 리스트
- 예상 경비 표시
- 예산 사용률 표시

---

### 여행지 상세 페이지

선택한 여행지의 상세 정보 제공

- 여행 경비 상세
- 관광지 정보
- 추천 여행 동선

## 🔗 API 통신

Frontend는 REST API를 통해 여행 추천 데이터를 요청한다.

### 여행지 추천 API
POST /api/travel/recommend

### Request

```json
{
  "budget": 1000000,
  "days": 3,
  "people": 2
}
```
Response
[
  {
    "place": "오사카",
    "totalCost": 950000
  }
]
